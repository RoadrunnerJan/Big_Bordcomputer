\doxysubsection{Detailed Description}
Sensor value calculation and management implementation. 

Handles real-\/time processing of sensor values including low-\/pass filtering, range validation, error detection, and formatted output for display. Supports multiple sensor types with independent filtering and state management.

\item Formatted output string generation
