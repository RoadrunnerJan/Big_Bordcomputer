\doxysubsection{Detailed Description}
Main application entry point for E36 Board Computer. 

Free\+RTOS-\/based multi-\/display gauge system with real-\/time sensor processing, LVGL graphical interface, and peripheral management for classic vehicle dashboard.

ESP-\/\+IDF application main entry point. 

Initializes all system resources and creates the main LVGL tick task for periodic display updates. After task creation, enters idle mode while background tasks handle application logic.

void init\+\_\+system (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Initialize all hardware and system components. 

Sets up all peripherals and services required for the board computer:\+
\begin{DoxyItemize}
\item Display backlight PWM and brightness control
\item ADC for sensor input
\item I2C bus with RTC (DS3231) and synchronizes system time
\item Time adjustment buttons with callbacks
\item SPI display layers and LVGL graphics engine
\item PWM/\+\+ADC sensor mode selector
\item Optional buzzer component for temperature alerts
\end{DoxyItemize}

Must be called once during application startup before creating tasks.
