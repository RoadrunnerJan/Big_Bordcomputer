This project implements a custom-\/built boardcomputer for classic vehicles like the BMW E36. Instead of the standard BMW boardcomputer (which only displays average fuel consumption or broken light bulbs), or multifunctional clocks (date, time, temperature), this system provides a complete display of current engine parameters.

{\bfseries{Main Features:\+}}
\begin{DoxyItemize}
\item Display of oil temperature, oil pressure, battery voltage, and time/\+temperature
\item Four round TFT displays for separate gauges
\item Adjustable number and type of displays
\item Fully customizable design via LVGL
\end{DoxyItemize}

The E36 version uses four displays, each with its own gauge. All parameters are configurable.\hypertarget{index_autotoc_md1}{}\doxysubsection{\texorpdfstring{Hardware Overview}{Hardware Overview}}\label{index_autotoc_md1}
The project is based on a powerful ESP32-\/\+P4 that controls all peripherals.

{\bfseries{Components used for the E36 boardcomputer:\+}}
\begin{DoxyItemize}
\item {\bfseries{ESP32-\/\+P4 WT9932\+P4-\/\+TINY}} -\/ Main processor
\item {\bfseries{4x 1.\+28 Inch Round TFT LCD Display (240x240 RGB) with GC9\+A01 Driver}} (\href{https://de.aliexpress.com/item/1005009364283361.html}{\texttt{Link to Ali\+Express}})
\item {\bfseries{Custom PCB}} to connect the displays and peripherals with the ESP (in the {\ttfamily hardware/\+} directory)
\item {\bfseries{5V Buzzer}} for warnings when outside temperature is below 3°C
\item {\bfseries{2x Buttons}} for time adjustment
\item {\bfseries{12V to 5V Converter}} for power supply
\item {\bfseries{BMW Oil Temperature and Pressure Sensor (Hella 6PP 010 378-\/201)}} -\/ PWM signal from engine
\item or using analog {\bfseries{VDO sensors}} for oil pressure and temperature
\end{DoxyItemize}

The datasheets for the used components can be found in the {\ttfamily datasheets/\+} directory.\hypertarget{index_autotoc_md2}{}\doxysubsection{\texorpdfstring{Software}{Software}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Free\+RTOS Tasks}} for scheduling different operations
\item {\bfseries{LVGL}} for designing the gauges
\item {\bfseries{ADC Sampling}} of sensor values or alternatively {\bfseries{PWM Sampling}} of oil pressure and oil temperature based on VDO sensor characteristics; PWM sampling is based on Hella Sensor 6PP 010 378-\/201
\item {\bfseries{Modular Architecture}} with separate directories for calculation, logging, LVGL-\/\+UI, and peripherals
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysubsection{\texorpdfstring{How to Flash the Project}{How to Flash the Project}}\label{index_autotoc_md3}
