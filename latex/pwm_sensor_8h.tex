\doxysubsection{Detailed Description}
PWM sensor reader for Hella 6PP 010 378-\/201 oil sensor. 

Decodes PWM pulse signals for oil pressure and temperature readings using MCPWM (Motor Control PWM) capture functionality. Supports multi-\/pulse sensors with diagnostic feedback.
