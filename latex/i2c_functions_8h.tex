\doxysubsection{Detailed Description}
I2C interface for DS3231 RTC and ADS1115 ADC. 

Handles I2C communication with DS3231 RTC and ADS1115 ADC sensors.

\begin{DoxyAuthor}{Author}
Jan Niklas Rodewald (JRO) 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
01.\+04.\+2026
\end{DoxyDate}
\begin{DoxyNote}{Note}
CHANGELOG v1.\+0 (01.\+04.\+2026) -\/ Initial implementation
\begin{DoxyItemize}
\item DS3231 real-\/time clock interface
\item ADS1115 ADC for analog sensor readings
\item NTC temperature table interpolation
\item Button control for time adjustment
\item Dual ADC support for enhanced sensor coverage 
\end{DoxyItemize}
\end{DoxyNote}
