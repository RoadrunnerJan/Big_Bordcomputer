The intention of this project is to build a customizable Bordcomputer for old vehicles like the BMW E36. Normaly this car could have a boardcomputer providing by BMW which shows something like the average gasoline consumption or broke light bulbs. Another option is a multifunctional clock which shows the date, time and temperature. There is no possibility to show the current oil temperature or pressure.

The aim of this project is to replace the standard bordcomputer by an own build computer which shows the actual oil temperature, pressure, voltage and the clock/\+termperature.

The version for the E36 is build with four displays. Each shows an own gauge for the measurements. The number of gauges is adjustable. The amount or type of display can get adjusted. The design can get adjusted, too.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Overall Hardware:\+}{Overall Hardware:\+}}\label{md__r_e_a_d_m_e_autotoc_md1}
The project uses a fast ESP32\+P4 which is connected to all the periphers. The following elements are used for the E36 boardcomputer:\+
\begin{DoxyItemize}
\item ESP32\+P4 -\/ WT9932\+P4-\/\+TINY
\item 4x 1.\+28 Inch Round TFT LCD Display Module (240x240 RGB) with GC9\+A01 Driver (\href{https://de.aliexpress.com/item/1005009364283361.html?spm=a2g0o.order_list.order_list_main.10.2d425c5foxNjLr&gatewayAdapt=glo2deu}{\texttt{https:\+/\+/\+de.\+aliexpress.\+com/\+item/\+1005009364283361.\+html?spm=a2g0o.\+order\+\_\+list.\+order\+\_\+list\+\_\+main.\+10.\+2d425c5fox\+Nj\+Lr\&gateway\+Adapt=glo2deu}})
\item Customized PCB Board to connect the Displays and the other periphery with the ESP (Uploaded in hardware directory of this project)
\item 5V buzzer to idenicate the outdoor temperature under 3°C
\item 2x buttons for changing the current time
\item 12v to 5v converter
\item BMW oil temperature and pressure sensor using a pwm signal to send the sensor values (Hella 6PP 010 378-\/201) -\/ connected to the engine
\end{DoxyItemize}
