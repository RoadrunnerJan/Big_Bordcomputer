\doxysubsection{Detailed Description}
Test simulation module for debug mode value generation. 

Generates deterministic test sensor values for development and testing without hardware. Useful for UI/\+logic testing when hardware sensors are unavailable.

\begin{DoxyAuthor}{Author}
Jan Niklas Rodewald (JRO) 
\end{DoxyAuthor}

\end{DoxyDate}
\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item Test value generation for all sensor types
\item Brightness and night mode simulation
\item Compile-\/time enable/\+disable support 
\end{DoxyItemize}
\end{DoxyNote}
