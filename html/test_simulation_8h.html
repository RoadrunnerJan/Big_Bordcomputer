Detailed Description

Test simulation module for debug mode value generation.

Generates deterministic test sensor values for development and testing without hardware. Useful for UI/logic testing when hardware sensors are unavailable.

Author
Jan Niklas Rodewald (JRO)
Date
01.04.2026
Note
CHANGELOG v1.0 (01.04.2026) - Initial implementation
  • Test value generation for all sensor types
  • Brightness and night mode simulation
  • Compile-time enable/disable support
