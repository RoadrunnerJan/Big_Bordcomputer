◆ get_i2c_adc_reference_voltage()

void get_i2c_adc_reference_voltage ( void )

Measure and update the ADC reference voltage dynamically.

Reads the reference voltage from the second ADS1115 device (if available) and updates the global reference_voltage variable for use in raw_to_res_safe(). If the measurement is out of valid range, defaults to ADC_ADS_REF_V.

Read reference voltage from ADS1115 ADC (Version 4.2 feature)
