◆ get_pwm_value()

double get_pwm_value ( int id)

Get the last decoded PWM value for a sensor by ID.

Retrieves the calculated sensor value (temperature, pressure, or diagnostic code) for the specified sensor ID. Returns -99 if value hasn't been updated.

Parameters
idPWM sensor ID (PWM_SENSOR_TEMP_PULSE_ID, PWM_SENSOR_PRES_PULSE_ID, or PWM_SENSOR_DIAG_PULSE_ID)
Returns
Calculated sensor value, or -99 if value not updated
