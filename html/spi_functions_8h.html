Detailed Description

SPI display driver for multiple LCD panels.

Manages SPI communication with multiple LCD displays using ESP-IDF and LVGL. Supports up to 4 independent 240x240 round TFT displays with GC9A01 drivers.

Author
Jan Niklas Rodewald (JRO)
Date
01.04.2026
Note
CHANGELOG v1.0 (01.04.2026) - Initial implementation
  • Multi-display SPI initialization (up to 4 displays)
  • LVGL integration for graphics rendering
  • Display task scheduling and timing
