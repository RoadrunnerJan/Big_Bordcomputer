Detailed Description

ADC interface for sensor readings via internal ADC.

Manages analog-to-digital converters from the microcontroller for reading various sensors: oil pressure, oil temperature, battery voltage, and outdoor temperature. Includes NTC thermistor and pressure table interpolation with calibration.

Author
Jan Niklas Rodewald (JRO)
Date
01.04.2026
Note
CHANGELOG v1.0 (01.04.2026) - Initial implementation
  • NTC thermistor table interpolation
  • Pressure sensor calibration table
  • Voltage divider calculations with dynamic reference

float raw_to_res_safe ( float raw_voltage,
float r_pullup )

Convert raw ADC voltage to sensor resistance with safety checks.

Calculates sensor resistance from measured voltage using voltage divider formula. Detects sensor errors (open circuit, short circuit) and returns ADC_FAIL_VALUE. Accounts for 100Ω protective series resistor in sensor circuit.

Parameters
raw_voltageRaw ADC reading in millivolts
r_pullupPull-up resistor value in Ohms
Returns
Calculated sensor resistance in Ohms, or ADC_FAIL_VALUE on error
